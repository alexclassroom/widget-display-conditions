Manages widget display by rules.

== Description ==

With an easy to use interface you can control on which page you want a particular widget to be displayed.

Following built-in rule parameters are available:

- post
- post type
- post term
- post type archive
- post template
- page
- page type
- page parent
- page template
 
== Installation ==

1. Upload `widget-display-rules` plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
